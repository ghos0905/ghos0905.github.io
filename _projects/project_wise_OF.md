`생체 센싱 시스템을 분석 및 응용하여 Surveillance 카메라 시스템에서 움직이는 객체를 인식하는 기술 개발 (2021.9 ~ 2022.4)`


Role:

움직이는 객체 인식 및 추적 경량화 알고리즘 개발(C++, OpenCV)


Action to Result 1:

라즈베리파이와 같은 저전력PC에서 돌리기 위해 OpenCV 라이브러리에서 Sparse Optical Flow 알고리즘 활용
공사장과 같이 카메라 진동에 의한 노이즈를 제거하기 위하여 Moving Window 방식 응용


Action to Result 2:

Sparse Optical Flow를 이용한 객체 탐지에서 Feature Point가 사라지는 경우를 대비하여 Feature Reset 기능 추가
장애물 뒤로 사라진 객체를 추정하기 위한 Memorized Estimation 알고리즘 개발
다양한 노이즈 환경에서 알고리즘의 Robustness 테스트 진행


Achievements:

기존 Optical Flow를 활용한 객체 인식 알고리즘 대비 인식률 향상
다양한 노이즈 상황에서도 안정적인 감지율과 연산처리 속도를 보임
연구 결과를 인정받아 해외 저널 출판 승인(Sensors)
