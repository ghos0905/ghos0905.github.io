SVM 카메라를 이용한 이동체 인식기술 개발

(주)와이즈오토모티브 산학협력 프로젝트, 2021.3~2022.3

버스용 SVM 영상합성영역에 존재하는 특정 객체(차량, 이륜차, 사람)를 식별하는 SW 개발
2021.3 ~ 2022.3



Role

  • 딥러닝 모델 학습환경 구축 (Pytorch, YOLOv5)
  • 학습 데이터 수집과 분석 방법 연구



Action to Result 1

  • 모델의 베이스 성능을 위하여 Public 데이터셋에서 필요한 객체 데이터 수집
  • 모델 결과 이미지를 분석 후 추가 데이터 수집 전략 설정
  • 추가 데이터를 수집 시간대, 실내외 환경, 카메라 위치 등의 특성에 따라 분류


Action to Result 2

  • 분류한 데이터를 타입 별로 상호 학습 및 평가 진행
  • 데이터 타입 간에 학습 영향력에 대한 Positive, Negative 관계 분석


Action to Result 3

  • 어둡거나 그늘이 진 부분, 카메라 왜곡이 심한 부분 등 특정 영역을 보완하는 타입의 데이터를 추가
  • 데이터 수를 조절해가면서 데이터 관계의 경향성과 효율적인 학습 데이터 수 분석



Achievements

  • 기업에서 원하는 타겟에 대한 인식률 목표 달성
  • 적은 데이터로도 효율적으로 목표 객체 인식률 향상하는 방법 제시
  • 프로젝트 수행 내용 관련 해외 학회 투고 및 우수 발표상 수상(ICEE 2022)
  • 해외 저널 투고 준비중
